# Boruvka

Реализация алгоритма Борувки для поиска минимального остовного дерева (MST)

## Файлы проекта

- `Main.java` - проверяет/генерирует тесты, читает графы, запускает алгоритм, пишет метрики в csv
- `Boruvka.java` - реализация алгоритма Борувки + счетчики времени и итераций
- `Graph.java` - граф (число вершин, ребер, массив ребер)
- `Edge.java` - ребро (src, dest, weight)
- `Subset.java` - элемент DSU (parent, rank) для компонент связности
- `Note.java` - вершина
- `tests.txt` - сгенерированные входные данные
- `metrics.csv` - результаты запусков

## Формат входных данных (`tests.txt`)

Каждый тест начинается строкой:

`#id;vertices;edges`

Далее идут строки ребер:

`src-dest-weight`

## Выходные метрики (`metrics.csv`)

Колонки:

`test_id,vertices,edges,iterations,time_ms,mst_weight`

## Запуск

1. Скомпилировать:

```bash
javac *.java
```

2. Запустить:

```bash
java Main
```

## Что измеряется

- `iterations` - количество итераций (внутренний счетчик алгоритма)
- `time_ms` - время работы алгоритма в миллисекундах (без генерации/чтения данных и прочего)
